## Installation

Stellen Sie sicher, dass Rust auf Ihrem System installiert ist. Nutzen Sie [rustup](https://rustup.rs/) zur Installation.

```bash
git clone https://github.com/yourusername/nodebridge.git
cd nodebridge
cargo build --release
```
